# Unity Mechanics Practice

This repository contains a Unity practice project created while following game development courses.

The goal of this project is to practice and understand basic gameplay mechanics, level setup, and interaction without focusing on complex scripts.

## Features

- Player character movement and basic controls
- Interactive doors activated by buttons
- Moving spike obstacles
- Jump pads / trampolines
- Simple level layout for gameplay testing

## Purpose

This project is **not a finished game**.  
It is a training playground used to experiment with Unity mechanics and scene composition.

## Engine

- Unity (URP)
